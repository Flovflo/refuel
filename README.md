# REFUEL - Smart Fuel Advisor

![iOS 17+](https://img.shields.io/badge/iOS-17%2B-blue)
![SwiftUI](https://img.shields.io/badge/SwiftUI-Framework-orange)
![SwiftData](https://img.shields.io/badge/SwiftData-Persistence-brightgreen)

Intelligent fuel station finder and consumption advisor for iOS. Helps users find the cheapest fuel and optimize their refill timing.

## Features
- Real-time government price data
- Smart consumption tracking
- City search
- Offline support (SwiftData)

## Setup
```bash
open refuel.xcodeproj
```
